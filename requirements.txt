# LLM Guard runs local models (prompt-injection, toxicity, PII/secrets,

# Heavy: pulls torch, transformers, spacy, presidio. CPU-only is fine.
# The default PyTorch CPU backend benchmarked faster than ONNX here, so ONNX is an

# cap is raised to <3.14 so it installs on Python 3.13 (upstream 0.3.16 caps at
# <3.13). Self-contained — no external fork. Mirrors pyproject.toml [tool.uv.sources].
